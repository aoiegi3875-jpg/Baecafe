# バランス調整と機能追加の確認（Walkthrough）

ここ数回のアップデートで行ったバランス調整、UI改善、および新規イベントシステムの実装内容をまとめました。

## 1. 経営バランスの超ハードコア化

*   **固定費の引き上げ**: 1日あたりの固定費を `5,000円` から `8,000円` に引き上げました。
*   **初期客数の減少**: 初期フォロワーを200人に減らし、来店人数の計算式を厳しくしました。これにより序盤は1日あたり数個しか売れません。
*   **鮮度の急激な劣化**: メニューが「古い」と判定されるまでの期間を短縮しました。
    *   0日目: 100%
    *   1日経過: 70%
    *   2日経過: 40% (売上半減)
    *   3日経過: 10% (廃棄寸前)
*   **攻略のコツ**: 初日に全資金を使ってでも3つのメニューを開発し、常に新しいメニューに入れ替える「自転車操業」が求められます。

## 2. メニュー生成時のAI評価調整

*   **「映え度」の二極化**:
    *   パンケーキ＋イチゴのような**「普通に美味しいメニュー」**は、AIが意図的に「映え度を低く（40点以下）」評価するようにプロンプトを調整しました。
    *   逆にタバスコや奇抜なコンセプトを混ぜた**「とんちきメニュー」**は、「映え度」が非常に高くなり（80点以上）、SNSでのバズ（売上増加）と引き換えに炎上リスクが高まるようになりました。
*   **コンサルタントのアドバイス**:
    メニュー開発画面の上部に「家賃を払うには、奇抜な具材を組み合わせて"とんちきメニュー"を作って一攫千金を狙え」という明確なメタメッセージを表示するようにしました。

## 3. 本日のトレンド機能

*   **日替わりトレンド**: 毎日ランダムで「パンケーキ」や「イチゴ」などが今日のトレンドに選ばれます（ショーケース画面に表示されます）。
*   **トレンドボーナス**: トレンドに合致したメニューをショーケースに置くと、そのメニューの**売上が2.5倍、フォロワー獲得数が2倍**になります。家賃を払うための強力な救済措置になります。

## 4. 朝のランダムイベント機能

炎上イベントとは別に、「1日が始まる直前」に約35%の確率で様々なイベントが発生するようになりました。
（※各イベントには専用のポップアップダイアログが表示されます）

### 追加されたイベント一覧
*   📸 **有名インフルエンサー来店！ (Good)**: その日の売上（客数）が一時的に2倍になります。
*   💰 **謎のパトロン出現 (Good)**: 経営支援金として10,000円の臨時収入が得られます。
*   📺 **地元テレビの密着取材 (Good)**: その日の売上（客数）が1.5倍になります。
*   🌧️ **大荒れの天気 (Bad)**: 悪天候により、その日の売上（客数）が半減してしまいます。
*   ❄️ **冷蔵庫トラブル発生！ (Bad)**: ショーケースに並んでいるすべてのメニューの鮮度が一気に1日分落ちてしまいます。
*   🐭 **ネズミのイタズラ (Bad)**: 食材をダメにされ、5,000円の損失が発生します。
